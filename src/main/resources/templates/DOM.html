DOM은 HTML요소(태그)에서 수행할 수 있는 작업들의 모임

            [HTML 요소 찾기]
            .getElementById("id명") : 해당 id의 요소를 읽어온다.
            .getElementsByClassName("클래스명") : 해당 클래스의 요소를 읽어온다.
            .getElementsByTagName("태그명") : 해당 태그의 요소를 읽어온다.
            .innerHTML : 태그사이에 내용을 출력

            [해당 요소 변경]
            요소.innerHTML = 변경내용 : 태그사이에 내용을 출력
            요소.attribute = 값 : 태그의 속성값을 변경(배경색, 높이, 폭, 글씨체)
            요소.style.property = 새로운 스타일 : 태그의 새로운 스타일을 적용
            요소.setAttribute(속성명, 값) : 지정한 속성의 값을 변경

            [요소 추가 및 삭제]
            document.createElement(요소) : HTML에 지정된 요소를 생성
            document.removeChild(요소) : HTML에서 해당 요소를 삭제
            document.appendChild(요소) : HTML에서 해당 요소를 추가
            document.replaceChild(새로운내용, 이전내용) : HTML에서 해당 요소의 내용을 변경
            document.write(문자열) : HTML에 출력
